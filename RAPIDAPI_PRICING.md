**Model:** Freemium with usage-based tiers

---

## How to Add Pricing on RapidAPI:

1. **Go to your API dashboard:**
   - https://rapidapi.com/developer/onboarding
   - Navigate to **ebenova-legal-docs**

2. **Click "Edit API" → "Pricing" tab**

3. **For each tier:**
   - Click **"Add Pricing Plan"**
   - Enter plan name (BASIC, STARTER, GROWTH, ENTERPRISE, ULTRA)
   - Set price ($0, $29, $79, $299, $999)
   - Set quotas (API calls/month)
   - Set rate limits (requests/second, requests/day)
   - Select features to include
   - Set overage pricing

4. **Save and Publish**

## Pricing Strategy Notes:

**Why this structure works:**

1. **Free tier (BASIC):** Low barrier to entry, developers can test
   - 5 calls = enough to test 1-2 documents
   - Converts to paid when they need more

2. **Starter ($29):** Sweet spot for freelancers
   - 100 calls = ~3 documents/day
   - Affordable for solo practitioners

3. **Growth ($79):** For serious businesses
   - 500 calls = ~16 documents/day
   - White-labeling is key differentiator

4. **Enterprise ($299):** For SaaS platforms
   - 2,000 calls = ~66 documents/day
   - SLA and dedicated support justify price

5. **Ultra ($999):** For high-volume users
   - 10,000 calls = ~333 documents/day
   - Competes with hiring legal staff

**Competitive Advantages:**
- Cheaper than lawyer ($500+/document)
- Faster than traditional services
- Africa-focused (underserved market)
- MCP-native (AI agent ready)
